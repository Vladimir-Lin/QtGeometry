矩形度规换算


N平面上有个矩形上有四个点，分别为Z0,Z1,Z2,Z3。 其中：

Vx = Z1 - Z0
Vy = Z3 - Z0
N = Vx × Vy

在N平面上有个点P，将点P换算成以< Vx , Vy >作为流形度规的单位，计算 式如下：

S = P - Z0
SL = | S | , 为S向量的单位长度。
Lx = | Vx | , 为Vx向量的单位长度。
Ly = | Vy | , 为Vy向量的单位长度。
Fx = ( S ．Vx  ) /  ( SL ．Lx ) ，为S在Vx向量上的投影分量。
Fy = ( S ．Vy  ) /  ( SL ．Ly ) ，为S在Vy向量上的投影分量。
fx = ( Fx ．SL ) / Lx ，为S以Vx向量作为度规的量。
fy = ( Fy ．SL ) / Ly ，为S以Vy向量作为度规的量。

fx = ( S ．Vx  ) / (  Vx ．Vx  )
fy = ( S ．Vy  ) / (  Vy ．Vy  )

当
fx >= 0.0
fx <= 1.0
fy >= 0.0
fy <= 1.0
四个条件成立时，点P位于矩形之内。

计算式列表：

  1. Vx = Z1 - Z0
  2. Vy = Z3 - Z0
  3. S = P - Z0
  4. fx = ( S ．Vx  ) / (  Vx ．Vx  )
  5. fy = ( S ．Vy  ) / (  Vy ．Vy  )

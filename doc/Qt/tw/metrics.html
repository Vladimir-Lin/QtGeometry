矩形度規換算


N平面上有個矩形上有四個點，分別為Z0,Z1,Z2,Z3。 其中：

Vx = Z1 - Z0
Vy = Z3 - Z0
N = Vx × Vy

在N平面上有個點P，將點P換算成以< Vx , Vy >作為流形度規的單位，計算 式如下：

S = P - Z0
SL = | S | , 為S向量的單位長度。
Lx = | Vx | , 為Vx向量的單位長度。
Ly = | Vy | , 為Vy向量的單位長度。
Fx = ( S ．Vx  ) /  ( SL ．Lx ) ，為S在Vx向量上的投影分量。
Fy = ( S ．Vy  ) /  ( SL ．Ly ) ，為S在Vy向量上的投影分量。
fx = ( Fx ．SL ) / Lx ，為S以Vx向量作為度規的量。
fy = ( Fy ．SL ) / Ly ，為S以Vy向量作為度規的量。

fx = ( S ．Vx  ) / (  Vx ．Vx  )
fy = ( S ．Vy  ) / (  Vy ．Vy  )

當
fx >= 0.0
fx <= 1.0
fy >= 0.0
fy <= 1.0
四個條件成立時，點P位於矩形之內。

計算式列表：

  1. Vx = Z1 - Z0
  2. Vy = Z3 - Z0
  3. S = P - Z0
  4. fx = ( S ．Vx  ) / (  Vx ．Vx  )
  5. fy = ( S ．Vy  ) / (  Vy ．Vy  )

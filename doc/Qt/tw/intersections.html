矩形與向量的交點計算



N平面上有個矩形上有四個點，分別為Z0,Z1,Z2,Z3。 其中：

Vx = Z1 - Z0
Vy = Z3 - Z0
N = Vx × Vy

有兩個點P1,P2組成向量V，P =P1 + tV線段與平面N的交點計算如下：

平面方程為 a．x + b．y + c．z + d = 0

N = ( a , b , c )

Z0必落於平面N之上，因此 N ．Z0 + d = 0

d = - N ．Z0

而P與平面N的交點亦落於平面N上，因此 N ．P + d = 0

N．P - N ．Z0 = 0
V = ( P2  - P1 )
N．( P1 + tV ) = N．Z0
t．N．V = N．( Z0 - P1 )
t = N．( Z0 - P1 ) / N．V

計算式列表：
  1. Vx = Z1 - Z0
  2. Vy = Z3 - Z0
  3. V = P2  - P1
  4. W = Z0 - P1
  5. N = Vx × Vy
  6. t = N．W / N．V
  7. P =P1 + tV
